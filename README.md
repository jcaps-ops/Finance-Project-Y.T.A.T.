# Finance Tracker

This project was made for keeping track of finances

## How to Use the Project

1. Sign up
2. Input your finances, budget, etc. 
3. Input all your purchases
4. Look at the statistics

## Key Features

- Keeps track of: 
   - Finances
   - Budgets
   - Expenses
- Creates graphs

## Installation Instructions

1. Install bcrypt
2. Install matplotlib
